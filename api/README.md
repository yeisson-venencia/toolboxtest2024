## DEV APP

```
npm run dev
```

## RUN APP

```
npm run start
```

## TEST APP

```
npm run test
```

## CREATE DOCKER IMAGE

```
docker build -f Dockerfile -t [name:tag] .
```

## CREATE DOCKER CONTAINER

```
docker run -it –-rm  –name [container_name] -p [host_port]:[container_port] [image_id/image_tag]
```
